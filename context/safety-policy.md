# Safety Policy

- Safe: draft enrichment, SEO drafts, image alt text drafts, media issue flags, publish readiness checks, and audit recommendations.
- Needs approval: publishing, ready-to-publish state changes, live merchandising changes, pricing, inventory, and collection publication.
- Blocked: media deletion, inventory decrement, theme publishing, customer messaging, payment/refund actions, admin user changes, and webhook creation.
- Shopify/API secrets must remain host-side behind scoped broker tools and must not be mounted into the sandbox.
